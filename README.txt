 عروض ساجر - نسخة جاهزة

المحتويات:
- index.html      الواجهة الرئيسية + لوحة التحكم
- styles.css      التصميم الكامل
- app.js          منطق الإضافة والتعديل والحذف والتصدير والاستيراد والتنبيهات

طريقة التشغيل:
1) افتح ملف index.html مباشرة في المتصفح
أو
2) ارفع الملفات على GitHub Pages / Netlify / أي استضافة ثابتة

الخصائص الموجودة:
- عرض العروض للزوار
- لوحة تحكم داخلية
- إضافة / تعديل / حذف / إخفاء / إظهار العروض
- تاريخ بداية ونهاية
- تصنيفات
- بحث وفلترة
- بيانات تجريبية جاهزة
- تصدير واستيراد JSON
- إشعارات متصفح محلية

مهم:
- هذه النسخة تعمل محليًا باستخدام localStorage داخل المتصفح
- إذا أردت نسخة احترافية سحابية حقيقية مع تسجيل دخول وحفظ على الإنترنت وتنبيهات لجميع المستخدمين، فهذه تكون المرحلة التالية باستخدام Supabase أو Firebase
